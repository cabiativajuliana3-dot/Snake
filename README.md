# 🐍 Snake Arcade

Un juego clásico de la serpiente con estilo arcade y efectos de sonido.

## Características
- Interfaz retro con efectos de neón.
- Sonidos de inicio, comida y game over (WAV).
- Velocidad progresiva (aumenta al comer).
- Controles por teclado (flechas).

## Archivos del juego
- `index.html` – Página principal.
- `style.css` – Estilos visuales.
- `game.js` – Lógica completa del juego.
- `sounds/` – Carpeta con los efectos de sonido:
  - `start.wav` – Al iniciar el juego.
  - `eat.wav` – Al comer una fruta.
  - `gameover.wav` – Al perder.

## Cómo jugar
1. Abre `index.html` en tu navegador.
2. Presiona **ESPACIO** para comenzar.
3. Usa las **flechas del teclado** para mover la serpiente.
4. Come las frutas rojas para crecer y sumar puntos.
5. Evita chocar con las paredes o contigo mismo.

¡Disfruta!
